Fuck eAsistent

A project to get eAsistent premium features for free.

Why?

The main reason I started this project is because eAsistent locks all my grades behind a paywall like it's some kind of cash grab. It's ridiculous how they're basically forcing students to pay just to see their own progress. I'm honestly fed up with these monopolistic tactics that make something as basic as checking your grades a hassle unless you cough up money. So screw that — I'm making this project so I can actually see my grades without having to deal with their nonsense.

How to use?

Since this app requires you to give your eAsistent login info, I decided that the best way you can use it is to self host. The app still supports multiple tracked accounts but in order to create one you need an invite code from the admin. If you don't know how to host it just ask one of your friends that is more tech savvy.
To see how you can self host you can view the guide on GitHub.
